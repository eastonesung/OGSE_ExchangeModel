## Introduction
This repository includes codes for preprocessing and microstructure model fitting for diffusion MRI data acquired by oscilating gradient spin echo (OGSE) sequence using Connectome 2.0 scanner.

## Terms of use

If you use any codes from this repository in your research, please cite the following article:

D. Sung, K.-S. Chan, J. Gerold, et al., “Complementary Sensitivity of Fixed-Time and Fixed-Oscillation Regimes to Exchange and Structural Disorder in the Human Brain Revealed Using Oscillating-Gradient Diffusion MRI With Ultra-Strong Gradients,” Magnetic Resonance in Medicine95, no. 6 (2026): 3429–3444, https://doi.org/10.1002/mrm.70300.
